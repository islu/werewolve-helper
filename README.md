## 基本介紹

狼人殺小幫手是一個建立在 LINE 上的聊天機器人，主要功能是發放狼人殺遊戲中的身分牌。

### 怎麼使用?

#### 如果你是創建房間者，你可以

1. 開設房間
   - 選擇「9人預設」或「自訂」
     - 如果選擇「9人預設」，預設為（1預言 1女巫 1獵人 3平民 3狼人）
     - 如果選擇「自訂」可以自由設定人數及身分
2. 查看房間
     - 可以查看目前加入的人及身分
3. 再來一局
     - 維持上局設定並重新分配身分

#### 如果你是創建房間者，你也可以

1. 輸入房間號碼即可加入遊戲並查看角色
2. 再來一局時，重新輸入房間號碼可以查看身分
